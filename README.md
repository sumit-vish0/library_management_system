# 📚 Library Management System

A robust and clean web application built with **Django** to manage library operations efficiently. This project allows administrators to manage books and members, while users can browse and request books.

---

## 🚀 Features

- **User Authentication:** Secure login/signup system for users and administrators.
- **Book Management:** Admin can Add, Update, and Delete books (CRUD).
- **Dashboard:** Interactive dashboard for users to see available books.
- **Search & Filter:** Easily search books by title, author, or genre.

---

## 🛠️ Tech Stack

- **Backend:** Python, Django
- **Database:** SQLite (Development)
- **Frontend:** HTML5, CSS3, Bootstrap (or whatever you used)
